# E-commerce Project

A modern, responsive E-commerce website template built with HTML5 and CSS3. This project demonstrates a clean user interface for an online store, featuring various page layouts and a consistent design system.

## Features

- **Responsive Design**: Adapts to different screen sizes.
- **Home Page** (`index.html`): features a hero section, featured products, and latest revisions.
- **Shop Page** (`shop.html`): Displays product listings.
- **Categories** (`catagories.html`): Browse products by category.
- **About Us** (`about.html`): Information about the brand.
- **Contact** (`contact.html`): Contact form and information.
- **Login** (`login.html`): User authentication page.
- **Modern Styling**: Custom CSS (`style.css`) with hover effects and clean typography.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling, Flexbox/Grid for layout.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Navigate through the links to view different pages.

## Project Structure

- `index.html`: Main landing page.
- `style.css`: Main stylesheet.
- `images/`: Directory for project assets.
- `*.html`: Individual pages for the site.
